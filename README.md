# DogBreed - UIKit

An example UIKit project showing dogs 🐕

## Tech stack

- ✅ UIKit
- ✅ Swift
- ✅ async/await
- ✅ CoreData
- ✅ Caching
- ✅ MVVM
- ✅ Unit tests with XCTest
- ✅ Network service with mock support for testing.


## Product
- View dog breeds and favourite your own.
- iOS14+
